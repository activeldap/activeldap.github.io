ActiveLdap

Ruby用オブジェクト指向LDAPインターフェイス。

  • Copyright © 2004-2006 Will Drewry <will@alum.bu.edu>
  • Copyright © 2006-2013 Kouhei Sutou <kou@clear-code.com>

説明

’ActiveLdap’はLDAPライブラリをきれいなオブジェクト指向APIで利用するためのライブラリです。ActiveRecordにインスパイアされています。ActiveRecordほどきれいさや柔軟性がないかもしれませんが、ほとんど同じように新しいオブジェクトを作ったり操作したりできます。

For example and usage – read the document.

必要なもの

Ruby intepreter

One of them:

  • Ruby (1.9.3, 2.0.x, 2.1.x, 2.2.x, 2.3.0)
  • JRuby

See the above links for installation.

LDAP client

JRuby doesn’t need to install new library because JRuby has builtin LDAP support. Ruby users need one of them:

  • Ruby/LDAP
  • Net::LDAP

See the above links for installation.

ActiveModel

A toolkit for building modeling frameworks like Active Record and Active Resource.

注意

  • Only GSSAPI SASL support exists due to Ruby/LDAP limitations

Rails

See Rails (doc/text/rails.textile in the repository and on GitHub) page for Rails integration.
